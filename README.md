### Notes

- Python 3.6+ is required
- Minimum ternimal size is 73x19
- Tested on macOS Catalina (iTerm) & Windows 10 (CMD, PowerShell)

### Starting a server

```sh
$ python3 server.py
```

### Running a client

#### Localhost server

```sh
$ python3 client.py
```

#### Any server

```sh
$ python3 client.py 158.108.30.234
```
